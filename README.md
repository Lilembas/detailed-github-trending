2. **核心功能**  
- ✅ **标准化 UI 声明机制**：工具可通过 `ui://` 协议声明 HTML 界面资源，由客户端自动拉取并渲染；  
- ✅ **深度交互能力**：UI 可接收工具调用参数（通过 host 通知）、触发其他 MCP 工具调用、响应用户操作并实时更新；  
- ✅ **多框架开箱即用**：提供 React、Vue、Svelte、Preact、Solid 及原生 JavaScript 的示例模板与 SDK 支持；  
- ✅ **AI 编程代理友好**：内置 4 个 Agent Skills（`create-mcp-app`、`migrate-oai-app`、`add-app-to-server`、`convert-web-app`），支持 Claude Code 等 AI 编程助手一键生成/迁移/集成应用；  
- ✅ **全栈 SDK 分层设计**：  
　• `@modelcontextprotocol/ext-apps`：基础 App 类与消息通信（PostMessageTransport）；  
　• `@modelcontextprotocol/ext-apps/react`：React 专用 Hook（`useApp`、`useHostStyles` 等）；  
　• `@modelcontextprotocol/ext-apps/app-bridge`：供聊天客户端宿主（host）嵌入和管理 UI；  
　• `@modelcontextprotocol/ext-apps/server`：供 MCP 服务端注册带 UI 元数据的工具；  
- ✅ **丰富开箱即用示例**：涵盖地图（CesiumJS）、3D 渲染（Three.js）、着色器（ShaderToy）、乐谱（ABC）、维基图谱、客户分群、预算分配、系统监控、语音转录、视频资源、PDF 浏览、二维码生成等 20+ 实战场景；  
- ✅ **跨平台客户端支持**：明确适配 ChatGPT、Claude、VS Code、Goose、Postman、MCPJam 等主流 MCP 宿主，并持续扩展。

3. **技术栈**  
- **前端 UI 层**：基于标准 HTML/CSS/JS，支持任意前端框架（React/Vue/Svelte/Solid/Preact/Vanilla JS）；依赖 Web 标准沙盒 iframe 与 `postMessage` 实现安全跨域通信；  
- **SDK 语言**：TypeScript（强类型定义，完整 API 文档）；  
- **构建与发布**：npm 包管理（`@modelcontextprotocol/ext-apps` 及其子包）；CI/CD 基于 GitHub Actions；
